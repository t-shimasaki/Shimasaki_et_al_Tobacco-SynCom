# Shimasaki et al. – Nicotine root microbiota study

This repository contains the R scripts and source data used for statistical analyses and figure generation in the manuscript.

## Contents
- `01_phylogenetic_tree/` – metabolomics data analysis
- `02_metabolites_analysis/` – metabolomics data analysis
- `03_Arthrobacter_Growth/` – bacterial growth assays
- `04_Arthrobacter_tolerance/` – nicotine tolerance experiments
- `05_Arthrobacter_inoculation/` – plant inoculation assays in mono-association system
- `06_Arthrobacter_competition/` – in plant competition assay
- `07_SynCom_community_analysis/` – SynCom community analysis
- `08_SynCom_Arthrobacter_dPCR/` – Absolute quantification by dPCR
